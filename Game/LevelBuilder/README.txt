---------------------------- THE LEVEL BUILDER ----------------------------

The controls for the level builder are:

Num1 to Num3 (not numpad, just the normal keys)
    - Select the different tile types, which are as follows:
        [1] = Starting tile
        [2] = Path tile, the path that the squorbs will travel down
        [3] = Ending tile

Place one of these tiles down by left-clicking anywhere on the grid.
Right-click to delete any tile.

Hit CTRL+S to save your current level, it'll prompt you to type the level's name.
Once you've filled it in, hit enter to save the level to the /levels folder in
the sln directory.
